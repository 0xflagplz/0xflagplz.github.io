First get you .deb version file you want to install
> sudo apt install ./filename.deb
> cd /usr/lib/jvm
  • you should see the version you installed


Step 5:
Enter the following command to open the environment variables file.

> sudo nano /etc/environment


Step 6:
In the opened file, add the following bin folder to the existing PATH variable.

ADD ---> /usr/lib/jvm/jdk-<version>/bin
ADD AT BOTTOM --> JAVA_HOME="/usr/lib/jvm/jdk-<verison>"


Before:
PATH="/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/games:/usr/local/games"

AFTER:
PATH="/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/games:/usr/local/games:/usr/lib/jvm/jdk-<version>/bin"



Step 7:
Enter the following commands to inform the system about the Java's location.Â Depending on your JDK version, the paths can be different.
    • > sudo update-alternatives --install "/usr/bin/java" "java" "/usr/lib/jvm/jdk-16.0.2/bin/java" 0
    • > sudo update-alternatives --install "/usr/bin/javac" "javac" "/usr/lib/jvm/jdk-16.0.2/bin/javac" 0
    • > sudo update-alternatives --set java /usr/lib/jvm/jdk-16.0.2/bin/java
    • > sudo update-alternatives --set javac /usr/lib/jvm/jdk-16.0.2/bin/javac



Step 8:
To verify the setup enter the following commands and make sure that they print the location of java and javac as you have provided in the previous step.
    • > update-alternatives --list java
    • > update-alternatives --list javac
    • > java -version
