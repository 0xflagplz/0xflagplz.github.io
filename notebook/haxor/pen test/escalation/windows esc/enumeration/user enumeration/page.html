What user are you?
  • > whoami
What are your user privileges?
  • > whoami /priv
What are groups are you assigned to?
> whoami /groups

These commands are helpful to know which user account you need to go after
  • sometimes involves lateral movement


More user info (generally shows # of users)
  • > net user
  • > net user <username> (shows information about the user)

Group Enumeration (Doesn't work 100% of the time)
  • > net localgroup <-- show all local groups
  • ***Maybe you know of a local group already***
  • ***Using example group administrators***
    • >net localgroup administrators
