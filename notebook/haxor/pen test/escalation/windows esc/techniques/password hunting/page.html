Searching Directories for specific files/names
> findstr /si password
Can add to this search
  • > findstr /si password *.txt *.ini *.config
  • - searches for file names password, .txt, .ini, .config files

Wifi Passwords
enumerate:
  • > netsh wlan show profile

Get Cleartext Password:
  • > netsh wlan show profile <SSID> key=clear
One liner to extract all passwords from all access points:
> cls & echo. & for /f "tokens=4 delims=: " %a in ('netsh wlan show profiles ^| find "Profile "') do @echo off > nul & (netsh wlan show profiles name=%a key=clear | findstr "SSID Cipher Content" | find /v "Number" & echo.) & @echo on
