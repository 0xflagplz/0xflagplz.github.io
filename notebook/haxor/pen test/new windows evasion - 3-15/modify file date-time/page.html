https://github.com/0xvpr/vpr-midas

a command line tool that edits a file's 'time created' and 'time modified' to whatever the user wants

Docker ENV:
# Download repo
> git clone https://github.com/0xvpr/vpr-midas cd vpr-midas
# Create docker image
> chmod +x docker-build.sh && ./docker-build.sh
> chmod +x docker-start.sh && ./docker-start.sh

Build Instructions
> git clone https://github.com/0xvpr/vpr-midas
> cd vpr-midas make
# sudo make install # if you want it available globally (only recommended for WSL/MSYS environments)

Example Usage
> vpr-midas path/to/file1 path/to/file2 -c "19900101000000" --silent
