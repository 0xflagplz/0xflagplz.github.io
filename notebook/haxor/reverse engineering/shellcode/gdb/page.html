x/nfu address
  • - x/ is used to examine memory
  • - n --> how much memory are we going to look at (default 1)
  • - f -- > Display format (the default is x[hexadecimal]), i[machine instructions], s [null-terminated string]
      • - x is normally what we will be looking at
  • - u --> unit size
    • - b -- bytes
    • - h -- halfwords (2 bytes)
    • - w -- words (four bytes [default])
    • - g -- Giant Words (8 bytes)
