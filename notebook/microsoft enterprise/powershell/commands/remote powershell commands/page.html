Show remote computers name
  • >Invoke-Command -ComputerName Client10 -Credential Cyber\administrator -ScriptBlock {hostname}
  • - crediantal Cyber\administrator isn't necessary wen running on admin account
  • - the response to this command was
    • response> Client 10
Create Local user on remote client
  • > Invoke-Command -ComputerName Client10 -Credential Cyber\administrator -ScriptBlock {net user invoke_user Passw0rd123 /add}
    Want to verify the user was created?
    • > Invoke-Command -ComputerName Client10 -Credential Cyber\administrator -Script {net user}
