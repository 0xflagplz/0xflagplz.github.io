The main differences are that executable CMD files reside in the file system (usually in C:/Windows/System32) and users can work with the output of CMD-based programs



Powershell
  • -Task-based command-line interface and scipting environment
  • - Powerful functions called cmdlets
  • - Object-oriented
  • - supports batch and powershell commands
CMD
  • - Command-line interpreter for Windows(system executables)
  • - Older batch language with standard functions
  • - String-based
  • - Interprets only batch commands
