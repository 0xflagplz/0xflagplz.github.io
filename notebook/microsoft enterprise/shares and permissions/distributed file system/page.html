Distributed File System (DFS)
  • - provides a single hierarchy namespace
  • - group server shares into a logical namespaces
  • - replicates folder acreoss multiple servers and sites

DFS supports standalone or domain-based namespaces.












DFS topology data is stored in the active directory.

A DFS tree structure is constructed from one or more root targets and DFS links




Namespace Server
  • > Provides the namespace and can be a computer or DC
Namespace Root
  • > Shares all public filders and controls DFS settings
  • if the domain name is cyber.comand the name of the root is Public, the namespace path will be \\Cyber\Public
