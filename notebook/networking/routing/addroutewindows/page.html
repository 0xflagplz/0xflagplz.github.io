print the routing table
> route print

adding the route
> route ADD destination_networkMASK subnet_mask  gateway_ip metric_cost
> metric cost and subnet_mask are optional --> subnet can be includeed most of the time though

Example
> route ADD 192.168.35.0 MASK 255.255.255.0 192.168.0.2
  • destination_network = 192.168.35.0
  • subnet_mask = 255.255.255.0
  • gateway_ip = 192.168.0.2

adding a { -p } creates a persistant route
> route -p ADD 192.168.35.0 MASK 255.255.255.0 192.168.0.2

---------------------------------------------------------------------------
Deleting a route

> route delete <desitination_network>

Example:
> route delete 192.168.35.0
